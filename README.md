# 🧮 Simple Calculator using Python

A beginner-friendly Python project that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator takes user input, processes the selected operation, and displays the result instantly.

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🛡️ Division by zero handling
* 💻 Simple command-line interface
* 🎯 Easy to understand for beginners

---

## 🚀 Technologies Used

* Python 3
